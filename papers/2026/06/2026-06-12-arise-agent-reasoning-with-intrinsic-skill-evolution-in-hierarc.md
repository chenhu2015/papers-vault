# ARISE: Agent Reasoning with Intrinsic Skill Evolution in Hierarchical Reinforcement Learning

## Summary

ARISE introduces a hierarchical RL framework where a shared policy operates at two levels: a Manager that maintains a tiered skill library built by summarizing successful solution traces, and a Worker that generates responses conditioned on retrieved skills. A dedicated skill-generation rollout creates and validates new skills, while a hierarchical reward guides co-evolution of reasoning ability and library quality. ARISE consistently outperforms GRPO-family algorithms and memory-augmented baselines across seven benchmarks, with particularly strong OOD gains.

## Key Contributions

- **Hierarchical policy**: single model acts as both Manager (skill selection/generation) and Worker (response generation), sharing weights without inference overhead
- **Tiered skill library**: skills are generated by structured summarization of successful traces after execution, then retrieved to condition future rollouts before execution
- **Hierarchical reward design**: separate reward signals for skill quality and task performance, co-evolving the library and reasoning capability
- **Strong OOD generalization**: particularly notable gains on out-of-distribution benchmarks, demonstrating that accumulated skills transfer beyond training distribution

## Relevance

Directly extends the SIRI (Jun 8) skill internalization thread — where SIRI discover-validate-internalize skills within the agent itself, ARISE builds a persistent, evolving skill library with hierarchical Manager/Worker structure.
