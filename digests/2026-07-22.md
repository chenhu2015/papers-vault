BPO (0.90) introduces a structurally new rollout topology for the agentic RL thread. The key insight — that sandboxes are deterministic and snapshottable, so sibling trajectories can share prefixes — is orthogonal to all prior credit assignment work in the vault. The four levels of the credit assignment hierarchy (token: TACO/EGRSD/RL-ZVP, turn: BDA/SD-MAR, step: PRMs, branch: IGRPO/CoBA-RL) all operate within a fixed linear or tree rollout structure where each rollout is independent. BPO makes explicit that in agentic settings, this independence is unnecessary and costly: shared prefixes contain shared variance, and exploiting them is lossless (proven unbiased). This is a fundamental insight into the gap between RLHF-derived rollout topologies and what agent execution environments actually permit.

HPO (0.83) provides the semantic complement to BPO's structural variance reduction. Where BPO reduces variance by sharing prefix trajectories, HPO reduces variance by aggregating semantically equivalent (state, action) pairs into intent clusters. The Wasserstein distance signal connects to information geometry (Fisher-Rao in DyCo-RL; Riemannian PPO-Clip in RIPO) — a growing pattern of papers that replace Euclidean/scalar gradient signals with geometric distances in transformed spaces. HPO's "intent space" is a learned semantic latent; DyCo-RL's Fisher-Rao is applied in attention-head space; the underlying pattern is the same: transforming the space before measuring divergence gives a more semantically calibrated gradient signal.

GRPO web agent failure (0.79) provides the most practically useful diagnostic in the vault: before applying GRPO to an agent, check whether sampling outperforms greedy. This "headroom" principle retrospectively validates the empty searches throughout June–July for cases where GRPO seemed to underperform: tasks where the supervised baseline is strong (near-mastered) will not benefit from GRPO regardless of hyperparameter tuning. The scale-dependent effective rank coupling (4B vs. 8B) is a concrete empirical finding about where the theory of RL improving representations breaks down. The VLM setting (Set-of-Marks screenshot) also connects this failure analysis to the Jul 21 VLM RL faithfulness thread: visual faithfulness failure modes (FWS, CORA, SIVA-RL, DyCo-RL) may be partially explained by headroom gaps in the visual reasoning subspace.

DEPT (0.77) closes the confirmed open gap on self-play/adversarial training for reasoning LLMs. The evolution impasse failure mode — homogenized strategies → deterministic outcomes → gradient collapse — is the self-play analogue to the exploration collapse failures in single-agent RLVR (RL-ZVP targets zero-variance prompts; DEPT targets zero-entropy match outcomes). The dual-scale value baseline divergence as an impasse signal is structurally analogous to CoBA-RL's dynamic capability gain tracking: both papers monitor the *rate of change* of policy improvement rather than static performance, enabling adaptive intervention before gradient collapse occurs.

**New thread — agentic RL variance reduction and rollout topology (two methods now characterized):**
1. **Structural (prefix-sharing) variance reduction** (BPO): exploit sandbox determinism to share prefixes across sibling trajectories; variance reduction = prefix-explained fraction of return variance
2. **Semantic (intent-space) variance reduction** (HPO): project policy and hindsight into intent space; Wasserstein distance provides bounded-variance gradient signal by aggregating semantically equivalent (state, action) pairs

The "headroom" principle from today's paper unifies the first three: all three represent cases where the gap between sampling and greedy policy is insufficient to provide useful gradient signal.
